# Instapro

MVP социальной сети для обмена фотографиями

Ссылка: https://alexandra-kuznetsova33.github.io/webdev-cw-instapro/

## Реализованная функциональность 

- Интеграция списка постов с API (динамический рендер из массива `posts`)
- Страница добавления нового поста (описание + загрузка изображения)
- Лайки и дизлайки без перезагрузки страницы (с запросом к API и обновлением только списка постов)
- Страница постов конкретного пользователя (переход по клику на имя автора)
- Безопасность: экранирование пользовательского ввода, защита от XSS
- Раскрытие изображения на весь экран при клике на фото
- Сохранение авторизации в localStorage (переживает перезагрузку страницы)
